# AbleBridge Connect
### Мы хотим предоставить больше удобства и возможности для каждого используя недостотатки других решений

## Для незрячих
* AI , волонтеры описывает пространство впереди, а не только под ногами
* Читает таблички, меню, документы
* Помогает ориентироваться в помещении и на улице
* Предупреждает об опасностях
* Экстренное уведомление родственникам
* И главное, орентация в локальную среду

## Для слабослыщащих
* Превращает речь в текст в реальном времени
* Фильтрует шум, выделяет важную информацию
* Субтитры для разговоров и звонков
* Понимание, а не просто звук
* Работает в обычных ситуациях (магазин, больница, ЦОН)
